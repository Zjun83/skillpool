# Installation

## Requirements

- Python 3.11+
- pip or uv

## Install from PyPI

```bash
pip install skillpool
```

## Install from Source

```bash
git clone https://github.com/skillpool/skillpool.git
cd skillpool
pip install -e ".[dev]"
```

## Verify Installation

```bash
python -c "import skillpool; print(skillpool.__version__)"
# 4.1.0
```
